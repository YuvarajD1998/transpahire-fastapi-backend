# TranspaHire Profile Service

A production-ready FastAPI microservice for profile management and resume processing, designed to integrate with the existing NestJS/Prisma TranspaHire platform.

## Features

- **Resume Upload & Processing**: PDF/DOCX support with AI-powered parsing
- **File Storage**: S3 integration with local fallback
- **AI Integration**: Hugging Face models for resume parsing and critique
- **Subscription Gating**: Feature access based on subscription tiers
- **Multi-tenant Safe**: Profile isolation per user/organization
- **Production Ready**: Logging, error handling, rate limiting, CORS
